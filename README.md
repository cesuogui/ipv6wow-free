单机版的游戏服务端与客户端无需修改任何游戏配置

服务端的玩家: 请运行/server/start
  1. 输入正确的联机秘钥后就可以生成联机邀请码,并在后台启动服务
  
  2. 邀请码会自动复制到剪切板,把联机邀请码发送给加入游戏的玩家

  3. 之后启动游戏客户端登录游戏
